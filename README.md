#### 弹幕

#### 时间戳

`TIME[x:x]ONLINE[number]`

如 `TIME2:3ONLINE397`: 凌晨2:3, 人气397

#### 发送人数统计

每天结束时在文件结尾统计`SPEAKERNUM[x]`
